## Linux Terminal Multiplexer - Screen
## [Video Tutorial](https://www.youtube.com/watch?v=I4xVn6Io5Nw)

### List Screen
```sh
screen -ls
```

### Screen Create
```sh
screen -S smscreen
```

### Detach From Screen
```sh
ctrl + a + d
```

### Re-Attach to Screen
```sh
screen -ls
screen -r screen-number(numeric part)
```
